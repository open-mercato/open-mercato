`yarn mercato feature_toggles toggle-delete` soft-deletes an existing feature toggle.

## Usage

```bash
yarn mercato feature_toggles toggle-delete --identifier <id>
```

## Options

| Option | Description | Required | Default |
| :--- | :--- | :--- | :--- |
| `--identifier` | Unique identifier of the toggle to delete. | Yes | - |

## Behavior

- Marks the toggle as deleted in the database (soft delete).
- If successful, outputs: `âœ… Feature toggle deleted: <identifier>`.
- If the toggle is not found, outputs an error message.
