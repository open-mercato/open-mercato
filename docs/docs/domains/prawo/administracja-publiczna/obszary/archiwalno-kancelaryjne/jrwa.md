title: JRWA / wykaz akt (roboczo)
sidebar_position: 40

Miejsce na opis struktury JRWA, zasad klasyfikacji oraz wymagań walidacyjnych.

Do zebrania:
- Minimalny zestaw pól klasy (symbol, hasło, kategoria, okres, uwagi).
- Reguły: kto może przypisać klasę, kiedy można zmienić, jak audytować.
- Powiązanie klas z procesami i typami dokumentów.
