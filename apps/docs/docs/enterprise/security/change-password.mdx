---
title: Change Password
description: Hardened self-service password change flow for the enterprise security module.
---

The security module introduces a new password change flow with enhanced security policies. Users manage it from their profile under **Profile → Security**, alongside MFA enrollment and other account security settings. This replaces the previous flow from the `auth`/`core` modules.

The new flow requires the user to provide their current password before setting a new one, and enforces configurable password quality rules (minimum length, character requirements) displayed in real time as the user types.
