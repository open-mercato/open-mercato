Open Mercato Enterprise Edition adds capabilities designed for teams running high-volume operations with stricter governance requirements.

## What is included

- **Advanced collaboration controls** such as record locks and conflict handling for concurrent editors.
- **Enterprise-grade controls** for operating larger organizations with stronger safety rails.
- **Additional enterprise modules** that can be enabled on top of the OSS core.

## Feature availability

Enterprise features are available only in Enterprise Edition builds and require the related module to be enabled.

Start with [Record locks](./record-locks) to configure concurrent editing safeguards.
