## How to verify what is active in your app

Inspect generated registries in `apps/mercato/.mercato/generated/`:

- `injection-tables.generated.ts`
- `injection-widgets.generated.ts`
- `interceptors.generated.ts`
- `enrichers.generated.ts`
- `component-overrides.generated.ts`
- `events.generated.ts`

These files are the runtime truth for what is currently discovered and wired.
