# Sales — Commands

The sales module registers the following commands on the command bus. You can intercept them via command interceptors or dispatch them from your own module.

## Channels

| Command ID | Description |
|---|---|
| `sales.channels.create` | Create a sales channel |
| `sales.channels.update` | Update a sales channel |
| `sales.channels.delete` | Delete a sales channel |

## Documents

| Command ID | Description |
|---|---|
| `sales.document-addresses.create` | Add an address to a sales document |
| `sales.document-addresses.update` | Update an address on a sales document |
| `sales.document-addresses.delete` | Remove an address from a sales document |

## Notes

| Command ID | Description |
|---|---|
| `sales.notes.create` | Create a note on a sales document |
| `sales.notes.update` | Update a note |
| `sales.notes.delete` | Delete a note |

## Payments

| Command ID | Description |
|---|---|
| `sales.payments.create` | Record a payment |
| `sales.payments.update` | Update a payment |
| `sales.payments.delete` | Delete a payment |
| `sales.payment-methods.create` | Create a payment method |
| `sales.payment-methods.update` | Update a payment method |
| `sales.payment-methods.delete` | Delete a payment method |

## Returns

| Command ID | Description |
