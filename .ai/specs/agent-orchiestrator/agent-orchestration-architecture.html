Agent Orchestration Layer — Architecture

Deterministic Open Mercato workflow as the spine; agents invoked as steps; business rules dispose. Built on OM (adopt) with a thin agent layer (build). Runtimes are pluggable (Foundry, Bedrock, OpenAI, Vertex, in-house) via adapters — A2A is the cross-runtime standard.

Principle: LLM proposes → OM disposes — the workflow controls flow, never the model.
